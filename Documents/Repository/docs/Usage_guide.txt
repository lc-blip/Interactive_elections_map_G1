---Objetivo da Aplicação---

Esta aplicação permite a exploração e análise dos resultados das Eleições Autárquicas Portuguesas de 2021, com foco nos resultados das Câmaras Municipais. Através de uma interface gráfica desenvolvida em Python com Tkinter e Matplotlib, o utilizador pode navegar pelos distritos e municípios de Portugal, visualizando os resultados eleitorais de forma interativa e intuitiva.

---Visão Geral da Interface---

A interface gráfica está organizada em três componentes principais:

Mapa Interativo de Portugal
Representa os distritos ou municípios, permitindo a seleção direta através de cliques. As geometrias são carregadas a partir da base de dados e desenhadas dinamicamente no canvas da aplicação.

Painel de Resultados
Apresenta uma tabela com os resultados eleitorais do município selecionado, incluindo votos, mandatos e informação associada a cada partido.

Visualização Gráfica
Exibe gráficos de barras que facilitam a comparação visual dos resultados entre partidos.

---Navegação e Funcionalidades---

-Seleção Geográfica-

O utilizador inicia a aplicação com uma visão geral do mapa e ao clicar num distrito ou município, a aplicação identifica a região selecionada. Os resultados eleitorais correspondentes são automaticamente carregados a partir da base de dados.

-Exploração dos Resultados-

A tabela de resultados permite analisar o desempenho de cada partido no município selecionado. Os gráficos complementam a tabela, facilitando a interpretação dos dados e a comparação entre partidos.

-Atualização Dinâmica-

Sempre que uma nova região é selecionada a tabela de resultados é atualizada e o gráfico é redesenhado de acordo com os novos dados.

---Interpretação dos Resultados---

Votos: número total de votos obtidos por cada partido no município selecionado.
Mandatos: número de mandatos atribuídos a cada partido.
Gráficos: permitem identificar rapidamente o partido mais votado e a distribuição relativa dos votos.

---Exemplos de Utilização---

-Analisar um município específico-
Clicar diretamente sobre o município no mapa para visualizar os resultados eleitorais detalhados.

-Comparar partidos num município-
Utilizar o gráfico de barras ou circular para comparar visualmente os votos e mandatos atribuídos a cada partido.

-Exploração geográfica-
Navegar entre diferentes regiões para observar variações nos resultados eleitorais a nível nacional.

---Limitações Conhecidas---

A aplicação está limitada aos dados das Câmaras Municipais, não incluindo outros órgãos eleitorais.
A precisão do mapa depende da simplificação das geometrias utilizadas, privilegiando o desempenho da aplicação.
Não é suportada a edição ou exportação direta dos resultados a partir da interface gráfica.
